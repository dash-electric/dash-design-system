05 · Layer 0 · Foundation

The part we never
negotiate.

Type ramp, spacing, motion, a11y floor, semantic tokens, the canonical purple. Any change requires a Design Lead RFC. This is the layer that gives the platform a single voice.
5 CARDINAL RULES
01 · Existing production code is never modified — DS is additive.
02 · No external form libraries (no RHF / zod / TanStack Query).
03 · Audit trail mandatory for legal / financial fields.
04 · Dash Purple canonical hex — single source of truth.
05 · Mitra-facing voice = formal "Anda", not casual "kamu".
