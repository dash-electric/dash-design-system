# Theme — `travel` (Dash Travel)

**Status:** placeholder. Spin-up theme for Dash Travel product line.

## Intent

Calm, elegant, considered. Used by **booking flows**, **itinerary planners**,
**traveller notifications** — environments where the user is planning ahead,
often anxious about logistics, and the UI should reduce (not amplify) urgency.

Accent: **ocean blue `#0284c7`** (Tailwind sky-600). Hue chosen for: (1) low
arousal compared to ride/logistic warm accents, (2) travel-industry colour
language (sky, airline branding), (3) clean contrast with Dash Purple.

## When this theme applies

- Booking flow (search → review → pay)
- Itinerary detail / day-plan UI
- Traveller-facing notifications (boarding, check-in, voucher)
- Concierge-style support surfaces

## Notes

- Primary CTAs use `--primary-base` (Dash Purple) — family identity.
- `--theme-accent-*` for itinerary timeline strokes, info pills, "you're
  booked" confirmations, map waypoints.
- Voice override: longer-form, reassuring (see `voice-overrides.md`).
