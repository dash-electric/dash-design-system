# Voice overrides — `travel`

Deltas only. Base voice = Layer 0.

## Deltas

- **Reassuring cadence:** confirmations include outcome + next-step ("Booking
  Anda telah dikonfirmasi. Voucher dikirim ke email dalam 5 menit.").
- **Soften timing language:** "kurang lebih", "perkiraan" allowed on ETAs
  (travel ETAs are inherently uncertain, hedging is honest, not weasel).
- **Concierge tone on support:** "Kami bantu cek" instead of "Sistem akan
  memproses".

## Preserved

- Formal "Anda".
- No slang, no emoji.
- Semantic error red stays canonical.
- Currency / date formatting follows Layer 0 i18n rules.
