# Voice overrides — `ride`

Deltas only. Base voice = Layer 0 (`apps/docs/registry/rules/dash-ai-rules.md`
§ Voice + CLAUDE.md cardinal rule #5).

## Deltas

- **Pace:** slightly faster cadence; shorter sentences than default Dash voice
  because surface is consumed in-motion (driver glancing at phone between
  ride legs).
- **Imperatives:** prefer single-verb actions ("Terima", "Tolak", "Lanjutkan")
  over phrases ("Terima permintaan ini") on primary CTAs.
- **Status copy:** present-tense, no hedging ("Dispatch berjalan", not
  "Dispatch sedang diproses").

## Preserved (do NOT change)

- Formal "Anda" — mitra-facing UI formality (CLAUDE.md rule #5).
- No slang, no softeners ("yaa", "dong", "kok").
- No emoji in production UI strings.
