# Voice overrides — `logistic`

Deltas only. Base voice = Layer 0.

## Deltas

- **ID-first copy:** lead with transaction/AWB/batch ID before description.
  ("AWB-882104 — pickup belum dijemput", not "Pickup belum dijemput. AWB:
  AWB-882104").
- **Terse imperatives on batch actions:** "Scan", "Konfirmasi", "Pindai
  ulang" — no full sentences on warehouse-scanner UI.
- **Quantity-prominent:** counts before nouns ("12 paket menunggu" not
  "menunggu 12 paket").

## Preserved

- Formal "Anda" (mitra-facing).
- No slang. No emoji.
- Semantic error red stays canonical — orange accent is NOT error.
